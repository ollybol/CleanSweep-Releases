# CleanSweep Releases

This public repository hosts notarized CleanSweep release artifacts and the
signed Sparkle update feed.

- Download published builds from [Releases](https://github.com/ollybol/CleanSweep-Releases/releases).
- `appcast.xml` is authenticated with CleanSweep's embedded Sparkle EdDSA
  public key.
- This repository contains distribution metadata only; it does not contain the
  CleanSweep source code.
